# Neon Starfield Rave

An immersive, high-performance interactive starfield animation with intense neon rave aesthetics. Features rainbow gradient backgrounds, animated color-shifting stars with glowing trails, and real-time controls.

## Features

- **Ultra Neon Visuals**: Vibrant rainbow gradient background with pulsing colors
- **Animated Stars**: Color-shifting stars with glowing halos and comet-like trails
- **Interactive Controls**:
  - Star Count (50-1000)
  - Speed (0.1x-5x)
  - Trail Length (5-50 frames)
  - Star Size (1-10px)
  - Trail Fade (0.85-0.99)
- **Collapsible UI**: Hide/show controls with gear icon
- **Performance Optimized**: 60 FPS with efficient rendering
- **Responsive Design**: Works on desktop and mobile

## Live Demo

Open `index.html` in any modern web browser.

## Controls

- **Sliders**: Adjust animation parameters in real-time
- **Pause Button**: Freeze the animation
- **Reset Button**: Return to default settings
- **Spacebar**: Toggle pause/play
- **Gear Icon**: Collapse/expand control panel

## Technical Details

- Pure vanilla JavaScript (no dependencies)
- HTML5 Canvas rendering
- Optimized draw operations for smooth 60 FPS
- Debounced window resize handling
- Frame rate limiting for consistent performance

Works in all modern browsers:
